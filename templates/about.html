OverCTRL — минималистичный и функциональный веб-сервис для хранения заметок. Разработан с акцентом на простоту и эффективность. Интерфейс отсутствует прост и интуитивно понятен :D

Используемые технологии:

  • Backend: Python + Flask
  • Frontend: HTML + CSS
  • База данных: SQLite
