OverCTRL — тестовое веб-приложение, для хранения заметок. Позволяет добавлять, просматривать и удалять заметки. Интерфейс отсутствует прост и интуитивно понятен :D

Используемые технологии:

  • Backend: Python + Flask
  • Frontend: HTML + CSS (и немного Bootstrap'а)
  • База данных: SQLite
